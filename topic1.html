What is a ‘Web application?’

A web application presents dynamically tailored content based on request parameters, tracked user behaviors, and security considerations.

What does a Web Application do?

A web application enables us to distribute documents over the World Wide Web. It manages concurrency, enabling access to a single web-based resource by multiple users. It can generate dynamic content, building web pages on the fly from sources of data that may include data supplied by users. It can include declaratives, role-based security, which enables you to allow or deny to specific resources to users based on their user roles.

A web application provides facilities to connect to a database so that its content can be kept in permanent storage, to be retrieved when required. It utilizes the transactional services of a database so that updates to the content are reliable and consistent. It leverages the services of its underlying hardware and software infrastructure to run the same application across multiple machines, enabling scalability.
