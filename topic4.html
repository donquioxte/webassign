Hypertext Transfer Protocol

WWW uses HTTP to send information. When using this protocol, the domain name (or IP address) is preceded by ‘http://’ HTTP is a request-response protocol as will be seen under the section on web browsers. HTTP requests are always of a specific type; GET, POST, HEAD, PUT, DELETE, CONNECT, OPTIONS or TRACE.

Hypertext Markup Language

It represents textual and image content. It is platform independent. This means that the text and the content are encoded in a way that they can be displayed on a wide range of computers. This will be covered in details later in the course.
